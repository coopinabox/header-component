## how to install

```
git clone https://github.com/coopinabox/header-component
npm install
```

## how to develop

```
npm run develop
```

## how to test

```
npm test
```

- to run client tests, you need to install [`xvfb`](http://packages.debian.org/stable/xvfb)
  - `[sudo] apt-get install xvfb`
